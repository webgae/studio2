# **App Name**: BloggerNext

## Core Features:

- Fetch Posts: Retrieve posts from Blogger API using the provided API key and Blog ID.
- Display Post List: Show a list of blog posts on the main page with titles and excerpts.
- Individual Post View: Display a single blog post with full content and metadata (e.g., date, labels).
- Search Functionality: Implement a search bar to find posts by title or content.
- Label Filtering: Allow users to filter posts by labels/tags.

## Style Guidelines:

- Primary color: Light desaturated blue (#A7D1E8), reflecting the traditional color of blog sites.
- Background color: Very light blue (#F0F8FF). This is a lighter tint of the primary to create an airy, spacious feeling.
- Accent color: Slightly darker desaturated blue (#75A9D1) to draw attention to key UI elements such as interactive elements.
- Headline font: 'Playfair', serif, to convey an elegant feel. Body font: 'PT Sans', sans-serif, for comfortable reading. Note: currently only Google Fonts are supported.
- Use a clean, responsive layout that adapts to different screen sizes.
